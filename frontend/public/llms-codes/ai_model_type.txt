[AI] AI Model Types -- AI denotes a group of machine learning model types built on artificial-intelligence techniques such as neural networks, reinforcement learning, and symbolic reasoning. These models aim to emulate cognitive functions and are used for tasks like perception, decision-making, language generation, and autonomous control.
[AI.01] Supervised: Classification -- AI.01 refers to supervised learning models that predict categorical labels from input features, trained on labeled datasets where the correct output is known. These models learn a mapping from inputs to discrete classes and are evaluated using accuracy, precision, recall, and related metrics.

[AI.03] Unsupervised: Clustering -- AI.03 denotes unsupervised clustering models, which automatically group data points based on similarity without using labeled outcomes. These algorithms, such as K-means, hierarchical clustering and DBSCAN, discover inherent structures, support pattern identification, and reduce data complexity for further analysis.
[AI.04] Unsupervised: Dimensionality Reduction -- AI.04 identifies unsupervised machine-learning techniques that reduce the number of variables in a dataset while preserving essential structure, such as principal component analysis, t-SNE, and autoencoders. These methods transform high-dimensional data into lower-dimensional representations to simplify analysis, visualization, and downstream tasks.
[AI.05] Reinforcement Learning -- Reinforcement Learning is a type of machine learning where an agent learns to make decisions by interacting with an environment and receiving feedback in the form of rewards or penalties. It focuses on sequential decision-making, using trial-and-error to maximize cumulative reward over time.
[AI.06] Deep Learning: CNN -- AI.06 designates convolutional neural networks, a class of deep learning models that process data with a grid-like topology such as images. CNNs use layers of convolutional filters to automatically learn spatial hierarchies of features, enabling tasks like image classification, detection and segmentation.
[AI.07] Deep Learning: RNN/LSTM -- AI.07 covers recurrent neural network architectures, focusing on long short-term memory units that enable learning from sequential data by maintaining a memory cell to capture long-range dependencies, making them effective for tasks such as language modeling, speech recognition, and time-series prediction.
[AI.08] Deep Learning: Transformer -- AI.08 refers to transformer-based deep learning models, which process data using self-attention mechanisms to capture contextual relationships. [AI.10] Generative: VAE -- AI.10 designates generative models built with variational autoencoders, which use an encoder-decoder neural network to learn a probabilistic latent representation of data. By sampling this latent space, the model can synthesize new instances that resemble the training distribution, supporting tasks such as image creation, data augmentation, and unsupervised feature learning.
[AI.11] Generative: Diffusion Model -- AI.11 classifies diffusion models, a type of generative neural network that creates data by progressively denoising random noise to match a target distribution. These models learn the reverse diffusion process and are widely used for image synthesis, video generation, and other creative content tasks.

[AI.13] Graph Neural Network -- AI.13 refers to Graph Neural Networks, a class of machine learning models that operate directly on graph-structured data by aggregating information from nodes and their neighbors. They are used for tasks such as node classification, link prediction, and graph-level inference in domains like social networks, chemistry, and recommendation systems.
[AI.14] Multi-Modal Model -- AI.14 Multi-Modal Model refers to a machine learning architecture that simultaneously processes two or more distinct data types-such as text, images, audio, or video-to learn shared representations and perform tasks that require cross-modal understanding, like image captioning, visual question answering, or speech-driven image retrieval. These models typically combine separate modality-specific encoders with a fusion mechanism and are trained on datasets containing aligned multimodal examples.
[AI.15] Neuro-Symbolic -- Neuro-symbolic methods blend neural networks with symbolic reasoning, merging data-driven learning and rule-based logic. The approach seeks the perceptual power of deep learning while retaining the interpretability, compositionality, and knowledge manipulation strengths of symbolic AI.
[AI.16] Federated Learning -- Federated learning is a distributed machine-learning approach where multiple devices or servers collaboratively train a shared model while keeping raw data local. It enhances privacy and reduces bandwidth use by transmitting only model updates, and is used in settings such as mobile keyboards and healthcare data aggregation.
