[Y02] TECHNOLOGIES OR APPLICATIONS FOR MITIGATION OR ADAPTATION AGAINST CLIMATE CHANGE -- This class covers selected technologies, which control, reduce or prevent anthropogenic emissions of greenhouse gases [GHG], in the framework of the Kyoto Protocol and the Paris Agreement, and also technologies which allow adapting to the adverse effects of climate change. If appropriate, a document can receive more than one indexing code of this class.
