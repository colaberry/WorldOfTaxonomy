[3G] 3GPP Series -- The 3G entry in the 3GPP Technical Specification Series comprises the third-generation mobile telecommunications standards developed by the 3rd Generation Partnership Project. It defines radio access, core network architecture, and service protocols that enable higher data rates, improved spectral efficiency, and enhanced multimedia capabilities for cellular networks worldwide.
[3G.21] Series 21 - Requirements -- Series 21 in the 3GPP Technical Specification Series defines the general requirements that all subsequent 3GPP specifications must meet, covering performance, security, and interoperability criteria for mobile communications systems.
[3G.22] Series 22 - Service Requirements -- The 3GPP Technical Specification Series 22 defines the service requirements that guide the development of mobile communication systems, covering functional and performance expectations for user-visible services such as voice, messaging, multimedia, and emergency calls. It serves as a reference for designing network features and validating that emerging technologies meet agreed-upon service standards.
[3G.23] Series 23 - Architecture -- Series 23 of the 3GPP Technical Specification Series defines the overall system architecture for 3GPP networks, describing core network components, functional entities, and their interfaces. It establishes the structural framework that underpins later specifications for radio access, services, and protocol layers.
[3G.24] Series 24 - Signaling Protocols -- Series 24 of the 3GPP Technical Specification Series defines the signaling protocols used in mobile networks, covering the Non-Access Stratum, Radio Resource Control, and related procedures that manage connection setup, mobility, and session handling across GSM, UMTS, LTE and 5G systems.
[3G.25] Series 25 - Radio (UTRA) -- The 3GPP TS 25 series defines the radio access specifications for UTRA, the UMTS terrestrial radio system. It details the physical layer, including channel coding, modulation, multiplexing and radio channel characteristics, establishing the standards for air-interface operation and performance in 3G networks.
[3G.26] Series 26 - Codecs -- Series 26 in the 3GPP Technical Specification Series defines the audio and video codecs used in mobile communications, specifying their functional requirements, performance criteria, and interoperability guidelines. It covers codecs such as AMR, EVS, AAC, and H.264, enabling voice, video, and multimedia services across 3GPP networks.
[3G.27] Series 27 - Data Terminal -- Series 27 of the 3GPP Technical Specification Series defines the requirements and interface specifications for data terminal equipment that connects to mobile networks. It standardises command sets, signalling procedures and functional capabilities such as AT commands, data transfer modes and service handling to ensure interoperability between terminals and the network.
[3G.28] Series 28 - Charging -- Series 28 of the 3GPP Technical Specification Series defines the architecture, interfaces and procedures for charging in mobile telecommunications, covering both offline and online charging, real-time credit control, rating and settlement, as well as the related policy and QoS interactions. It provides the normative framework for how operators implement charging functions across the core network.
[3G.29] Series 29 - Core Network Protocols -- Series 3G.29 of the 3GPP Technical Specification Series defines the core network protocols used in mobile communications, detailing the architecture and functions of protocols such as SCTP, GTP, DIAMETER and related signaling procedures that enable connectivity between radio access networks and the EPC. It provides the normative requirements for protocol behavior, message formats and security aspects across 4G and 5G core networks.
[3G.31] Series 31 - USIM -- Series 31 of the 3GPP Technical Specification Series defines the Universal Subscriber Identity Module (USIM) architecture, interfaces and security mechanisms for mobile networks. It specifies functional and protocol requirements for USIM cards used in LTE and 5G devices, covering authentication, key management and application programming interfaces.
[3G.32] Series 32 - OAM -- Series 32 of the 3GPP Technical Specification Series defines the Operations, Administration and Maintenance (OAM) framework for mobile networks, covering protocols, procedures and functional entities that enable monitoring, fault management, performance measurement and configuration of both radio and core network elements.

[3G.34] Series 34 - UE Testing -- Series 34 in the 3GPP Technical Specification series defines the testing framework for user equipment, detailing test cases, procedures and performance criteria to verify conformance and interoperability of mobile devices with network standards. It guides manufacturers and test labs in assessing UE functionality, radio behavior and compliance across release versions.
[3G.35] Series 35 - Algorithms -- Series 35 of the 3GPP Technical Specification Series contains specifications for algorithms used in mobile communications, including encryption, integrity protection, and radio access network functions. It defines the requirements and test methods for these algorithms to ensure interoperability and security across 3GPP systems.
[3G.36] Series 36 - LTE Radio -- The 3GPP Technical Specification Series entry 3G.36 groups all LTE radio-access standards, defining the air-interface architecture, physical layer procedures, channel coding, modulation, and radio resource management. It provides the detailed specifications that enable interoperable LTE equipment and networks worldwide.
[3G.37] Series 37 - Radio Measurement -- Series 37 in the 3GPP Technical Specification series defines the framework for radio measurements used in mobile networks, covering measurement types, procedures and reporting formats that support handover, cell reselection and performance monitoring. It applies to LTE and NR systems and provides guidelines for both network equipment and user equipment implementations.
[3G.38] Series 38 - 5G NR Radio -- Series 38 of the 3GPP Technical Specification Series defines the radio interface for 5G New Radio (NR). It specifies the physical layer, channel structures, modulation, coding and procedures for both standalone and non-standalone deployments, enabling high-throughput, low-latency mobile communications.
