[MA] ATT&CK Tactics -- The MITRE ATT&CK Tactics component groups adversary behaviors into high-level objectives that guide the planning and execution of attacks. Each tactic represents a phase of the intrusion lifecycle, such as Initial Access, Persistence, or Exfiltration, and is used to map techniques and mitigations. The code MA identifies this entry within the classification system.
[MA.01] Reconnaissance -- MA.01 Reconnaissance denotes the initial tactical phase in which adversaries collect technical and contextual data about target systems, networks, personnel, and defenses. This information gathering supports planning of subsequent operations and may involve open-source research, network scanning, or social engineering to identify exploitable assets.

[MA.03] Initial Access -- MA.03 Initial Access refers to techniques adversaries use to gain foothold within a target environment. This includes exploiting public-facing applications, phishing, or using valid credentials to enter the network. The tactic marks the first step in the attack chain, enabling subsequent actions.
[MA.04] Execution -- The Execution tactic in MITRE ATT&CK represents how adversaries run code on a victim's system to carry out malicious actions. It covers techniques such as command-line interfaces, scripting, and abuse of trusted executables that enable the attacker to achieve their objectives.
[MA.05] Persistence -- Persistence is a MITRE ATT&CK tactic that focuses on techniques adversaries use to retain a foothold on a system across restarts, logouts, or credential changes. It includes methods such as creating scheduled tasks, modifying startup scripts, or installing services that automatically execute malicious code. These actions enable continued access and control despite defensive measures.
[MA.06] Privilege Escalation -- MA.06 Privilege Escalation is a MITRE ATT&CK tactic in which attackers attempt to gain higher-level permissions on a compromised system. They may exploit software vulnerabilities, abuse misconfigurations, or harvest credentials to move from limited to administrative rights, enabling broader impact and persistence.
[MA.07] Defense Evasion -- Defense Evasion (MA.07) refers to tactics adversaries employ to bypass security controls, hide their activities, and remain undetected within a victim environment. This includes modifying system logs, disabling security tools, using legitimate credentials, and obfuscating malicious code.

[MA.09] Discovery -- Discovery is a MITRE ATT&CK tactic that groups techniques adversaries employ to gather information about a target system or network. It includes actions like enumerating users, processes, services, and network configurations to map the environment and identify further attack opportunities.
[MA.10] Lateral Movement -- Lateral Movement is a MITRE ATT&CK tactic in which attackers pivot from an initially compromised system to additional hosts within the network to expand control and access valuable resources. It includes techniques such as using remote services, credential dumping, Pass-the-Hash, and exploiting trust relationships to traverse the environment while evading detection.
[MA.11] Collection -- The Collection tactic describes an adversary's effort to gather data from a compromised environment to support their objectives. It includes locating and extracting files, recording screens or keystrokes, and aggregating information that can later be used for exploitation or exfiltration. This activity prepares the attacker for subsequent stages such as command-and-control or data theft.
[MA.12] Command and Control -- MA.12 - Command and Control is the MITRE ATT&CK tactic describing how adversaries establish and maintain communications with compromised systems. It involves creating channels that allow remote instruction, data exfiltration, and coordination of further actions while often using covert or encrypted protocols to evade detection.
[MA.13] Exfiltration -- Exfiltration is the ATT&CK tactic focused on stealing data from target systems and moving it to locations controlled by the adversary. It includes techniques that package, encrypt, or otherwise conceal data before it is transmitted over the network, via removable media, or through cloud services, enabling the attacker to harvest valuable information for further use or resale.
